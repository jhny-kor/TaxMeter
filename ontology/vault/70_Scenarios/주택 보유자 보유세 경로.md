# 주택 보유자 보유세 경로

주택 보유자가 공시가격 합계액, 과세기준일, 재산세, 종합부동산세 구간을 순서대로 확인하는 경로입니다.

## 사용자 경로

1. **공시가격 확인** → [[40_Terms/공시가격|공시가격]]: 국토교통부 부동산공시가격 알리미에서 주택별 공시가격을 확인한 뒤 소유 주택을 합산합니다.
2. **과세기준일 확인** → [[10_Taxes/National/RealEstate/종합부동산세 과세기준일|종합부동산세 과세기준일]]: 종합부동산세는 매년 6월 1일 현재 보유 여부로 과세대상을 판정합니다.
3. **공제금액 차감** → [[10_Taxes/National/RealEstate/종합부동산세 공제금액|종합부동산세 공제금액]]: 주택 9억원, 1세대 1주택 12억원 등 과세유형별 공제금액을 먼저 차감합니다.
4. **세율 구간 선택** → [[10_Taxes/National/종합부동산세|종합부동산세]]: 공제 후 공정시장가액비율을 적용한 과세표준으로 주택 수·토지 유형별 세율표 구간을 선택합니다.
