# 청년 정책지원 탐색 경로

청년 사용자가 자산형성, 주거금융, 서민금융 지원의 나이·소득·재산·한도 기준을 비교하는 경로입니다.

## 사용자 경로

1. **자산형성 계좌** → [[30_Supports/청년미래적금|청년미래적금]]: 2026년 출시 예정 청년미래적금의 나이, 개인소득, 가구소득, 납입한도와 기여금 기준을 확인합니다.
2. **청년도약계좌** → [[30_Supports/청년도약계좌|청년도약계좌]]: 개인소득, 가구소득, 금융소득종합과세 제외 요건을 확인합니다.
3. **청년 서민금융** → [[30_Supports/햇살론유스|햇살론유스]]: 만 19~34세와 연소득 기준, 동일인 보증한도 및 용도별 한도를 확인합니다.
4. **전세자금보증** → [[30_Supports/무주택 청년 특례전세자금보증|무주택 청년 특례전세자금보증]]: 무주택 청년의 나이, 소득, 보증한도 기준을 확인합니다.
