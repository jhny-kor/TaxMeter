# 소상공인 정책자금·바우처 판단 경로

자영업자·소상공인이 운영자금, 바우처, 도약 지원사업을 찾을 때 사업자등록, 매출, 체납, 자금 목적을 순서대로 판단하는 경로입니다.

## 사용자 경로

1. **생활사건 분류** → [[80_LifeLanguage/Events/소상공인 운영자금·지원금|소상공인 운영자금·지원금]]: 사업자금·소상공인 지원금 표현을 정책자금과 바우처 후보로 분류합니다.
2. **소상공인 여부** → [[81_Rules/Eligibility/소상공인 기본 지원대상 판정|소상공인 기본 지원대상 판정]]: 정상 영업 중인 소상공인인지 확인합니다.
3. **세금 체납** → [[81_Rules/Eligibility/세금 체납 소상공인 지원 제한|세금 체납 소상공인 지원 제한]]: 정책자금 융자 제한 사유가 될 수 있는 체납 여부를 확인합니다.
4. **정책자금** → [[30_Supports/소상공인 정책자금|소상공인 정책자금]]: 운전자금, 대환대출, 재도전, 성장기반 자금 중 맞는 유형을 고릅니다.
5. **바우처** → [[30_Supports/소상공인 경영안정 바우처|소상공인 경영안정 바우처]]: 연 매출 1억400만원 미만이면 경영안정 바우처 후보를 확인합니다.
