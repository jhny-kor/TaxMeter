# 부동산 양도 경로

부동산을 양도한 사용자가 양도차익 계산, 기본공제, 예정·확정신고 기한을 확인하는 경로입니다.

## 사용자 경로

1. **양도차익 계산** → [[10_Taxes/National/CapitalGains/양도소득세 계산 흐름|양도소득세 계산 흐름]]: 양도가액, 취득가액, 필요경비와 공제를 거쳐 양도소득 과세표준을 계산합니다.
2. **양도소득세율 확인** → [[10_Taxes/National/양도소득세|양도소득세]]: 기본세율과 자산 유형별 특례세율을 분리해 산출세액을 확인합니다.
3. **예정신고** → [[50_Deadlines/양도소득세 예정신고|양도소득세 예정신고]]: 토지·건물 등은 양도일이 속하는 달의 말일부터 2개월 이내 예정신고 여부를 확인합니다.
4. **확정신고** → [[50_Deadlines/양도소득세 확정신고|양도소득세 확정신고]]: 복수 양도 등 확정신고 대상은 다음연도 5월 신고 경로로 연결합니다.
