# 상속·증여 신고 경로

상속인 또는 수증자가 과세표준, 세율, 신고기한을 순서대로 확인하는 경로입니다.

## 사용자 경로

1. **상속·증여 유형 구분** → [[10_Taxes/National/상속세와 증여세|상속세와 증여세]]: 사망에 따른 이전인지 생전 무상이전인지에 따라 납세의무자와 신고기한이 달라집니다.
2. **세율표 확인** → [[10_Taxes/National/상속세와 증여세|상속세와 증여세]]: 상속세와 증여세는 같은 5단계 초과누진세율 구조를 사용합니다.
3. **상속세 신고기한** → [[50_Deadlines/상속세 신고|상속세 신고]]: 거주자 6개월, 비거주자 9개월 기한을 상속개시일이 속하는 달의 말일부터 계산합니다.
4. **증여세 신고기한** → [[50_Deadlines/증여세 신고|증여세 신고]]: 일반 증여는 증여받은 날이 속하는 달의 말일부터 3개월 이내 신고합니다.
