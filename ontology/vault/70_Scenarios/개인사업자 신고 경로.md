# 개인사업자 신고 경로

개인사업자가 사업자등록, 부가가치세 과세유형, 원천세, 종합소득세 신고를 순서대로 확인하는 경로입니다.

## 사용자 경로

1. **사업자등록** → [[60_Business/사업자등록 신청|사업자등록 신청]]: 사업 개시 전 또는 개시일부터 20일 이내 등록하고 과세유형 선택을 확인합니다.
2. **부가가치세 과세유형** → [[10_Taxes/National/부가가치세|부가가치세]]: 일반과세자·간이과세자 기준, 납부면제, 예정신고 예외를 매출 기준으로 판정합니다.
3. **원천세 반복 신고** → [[50_Deadlines/원천세 신고 납부 절차|원천세 신고 납부 절차]]: 인건비나 사업소득 지급이 있으면 다음 달 10일 또는 반기별 납부 반복 규칙을 관리합니다.
4. **종합소득세 확정신고** → [[50_Deadlines/종합소득세 확정신고|종합소득세 확정신고]]: 사업소득을 포함한 종합소득은 다음연도 5월 확정신고 경로로 연결합니다.
