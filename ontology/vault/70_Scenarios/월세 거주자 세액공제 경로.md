# 월세 거주자 세액공제 경로

자취방·원룸 월세를 낸 근로자가 월세액 세액공제 적용 가능성과 필요서류를 판단하는 경로입니다.

## 최신성 메타데이터

- **기준연도**: 2026
- **확인일**: 2026-05-04
- **폐지 여부**: active
- **개정 예정 여부**: none_announced
- **출처 URL**: https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?cntntsId=239025
- **출처 확인일**: 2026-05-02 확인

## 사용자 경로

1. **생활어 입력** → [[80_LifeLanguage/Expenses/월세·방값|월세·방값]]: 월세·방값 표현을 월세액 세액공제 후보로 매핑합니다.
2. **공식 항목** → [[20_Deductions/TaxCredits/월세액 세액공제|월세액 세액공제]]: 총급여, 종합소득금액, 주택, 공제율, 한도를 확인합니다.
3. **주소·무주택** → [[81_Rules/Eligibility/월세 주소 일치 요건|월세 주소 일치 요건]]: 전입신고와 주소 일치, 무주택 세대 요건을 확인합니다.
4. **필요서류** → [[82_Documents/월세 지급 증빙|월세 지급 증빙]]: 주민등록표등본, 임대차계약서, 지급 증빙을 준비합니다.
