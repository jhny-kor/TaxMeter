# 청년월세 현금성 월세지원 중복 제한

국토부 또는 지자체 현금성 월세지원사업을 이미 받고 있거나 청년월세 한시 특별지원을 24개월 수혜한 경우 제외대상인지 확인하는 중복 제한 규칙입니다.
