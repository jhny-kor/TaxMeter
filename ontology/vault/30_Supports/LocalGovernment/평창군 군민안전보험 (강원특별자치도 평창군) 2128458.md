# 평창군 군민안전보험 (강원특별자치도 평창군)

강원특별자치도 평창군 관할 지자체 지원금입니다. 평창군민이 예기치 못한 사고를 당했을 때 의료비와 보상금을 지원받을 수 있도록 마련된 제도입니다. 특히 자전거·교통사고, 재난사고 등에서 최대 수천만 원까지 보장이 가능해 생활 속 위험으로부터 군민을 지켜줍니다. 별도의 비용 부담 없이 전 군민이 자동 가입됩니다.
