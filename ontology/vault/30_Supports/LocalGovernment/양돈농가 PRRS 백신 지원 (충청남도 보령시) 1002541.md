# 양돈농가 PRRS 백신 지원 (충청남도 보령시)

충청남도 보령시 관할 지자체 지원금입니다. 돼지 생식기호흡기증후군(PRRS) 예방을 위해 백신 구입비를 지원해요. 질병 예방과 건강한 양돈 환경 유지에 도움이 됩니다.
