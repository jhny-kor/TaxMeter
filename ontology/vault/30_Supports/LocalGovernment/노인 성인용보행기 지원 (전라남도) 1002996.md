# 노인 성인용보행기 지원 (전라남도)

전라남도 관할 지자체 지원금입니다. 거동이 불편한 노인이 안전하게 이동할 수 있도록 성인용 보행기를 지원합니다. 이동 편의성을 높이고 일상생활 독립성을 강화합니다.
