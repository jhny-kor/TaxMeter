# 친환경농산물 인증수수료 지원사업 (경상남도 창녕군)

경상남도 창녕군 관할 지자체 지원금입니다. 농가가 친환경농산물 인증을 받을 때 발생하는 수수료를 지원하여 농가의 인증 비용 부담을 줄이고 안전한 농산물 생산을 돕습니다.
