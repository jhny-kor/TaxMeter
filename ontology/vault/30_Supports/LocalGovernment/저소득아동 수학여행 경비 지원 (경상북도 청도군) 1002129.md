# 저소득아동 수학여행 경비 지원 (경상북도 청도군)

경상북도 청도군 관할 지자체 지원금입니다. 저소득 가정 아동이 수학여행을 안전하게 다녀올 수 있도록 여행 경비를 지원합니다. 교육 기회 확대와 체험 학습 참여에 도움됩니다.
