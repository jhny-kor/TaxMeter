# 농업용 관리기 지원 (강원특별자치도 영월군)

강원특별자치도 영월군 관할 지자체 지원금입니다. 소규모 농가의 농작업 효율을 높이기 위해 관리기 구입 비용을 지원합니다. 농가당 연 1대, 최대 200만 원까지 지원되며, 노동력 절감과 생산성 향상에 실질적인 도움이 됩니다.
