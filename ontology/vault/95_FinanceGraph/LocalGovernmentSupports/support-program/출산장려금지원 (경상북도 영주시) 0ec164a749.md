# 출산장려금지원 (경상북도 영주시)

경상북도 영주시 관할 지자체 지원금입니다. 출산 가정의 경제적 부담을 덜고, 건강한 양육 환경을 조성하기 위해 신생아 출생 시 일정 금액을 지급합니다. 출산을 축하하고 초기 양육비 부담을 완화하여 가족이 안정적으로 생활할 수 있도록 돕습니다. 금액은 출생 아동 1인당 정액 지급되며, 출생신고 후 바로 신청 가능합니다.
