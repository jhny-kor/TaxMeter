# 저소득층 아동급식 평일·방학 지원 (부산광역시 수영구)

부산광역시 수영구 관할 지자체 지원금입니다. 저소득층 아동의 균형 잡힌 식사를 지원하여 건강한 성장과 학습 환경을 제공하는 제도입니다.
