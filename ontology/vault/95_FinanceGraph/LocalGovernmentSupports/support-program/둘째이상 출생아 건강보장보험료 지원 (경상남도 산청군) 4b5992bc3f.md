# 둘째이상 출생아 건강보장보험료 지원 (경상남도 산청군)

경상남도 산청군 관할 지자체 지원금입니다. 둘째 이상 자녀를 둔 가정의 건강보험료 일부를 지원하여 출산과 육아 부담을 경감합니다.
