# 한부모가족 자립지원 (경상남도)

경상남도 관할 지자체 지원금입니다. 한부모 가정이 안정적으로 생활하고 자립할 수 있도록 생활비, 교육비 등을 지원합니다. 연간 최대 60만 원 지원으로 경제적 부담을 줄이고 생활 안정에 도움을 줍니다.
