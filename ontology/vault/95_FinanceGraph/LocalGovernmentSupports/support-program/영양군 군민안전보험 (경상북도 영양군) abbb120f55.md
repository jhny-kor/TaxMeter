# 영양군 군민안전보험 (경상북도 영양군)

경상북도 영양군 관할 지자체 지원금입니다. 영양군 주민이 재난, 사고, 질병 등 긴급 상황에서 경제적 보호를 받을 수 있도록 지원합니다. 
보장 항목별 보험금 지급되며, 연간 자동 갱신됩니다. 
주민은 사고 발생 시 치료비와 생활비 부담을 줄이고, 안전망을 확보할 수 있습니다.
