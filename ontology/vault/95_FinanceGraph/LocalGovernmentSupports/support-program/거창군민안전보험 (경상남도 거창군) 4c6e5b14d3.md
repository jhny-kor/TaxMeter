# 거창군민안전보험 (경상남도 거창군)

경상남도 거창군 관할 지자체 지원금입니다. 거창군에 주소를 둔 모든 군민을 대상으로 예기치 못한 사고나 재난 발생 시 보험금을 지원합니다. 군민 누구나 자동 가입되며, 타 보험과 중복 보상도 가능합니다.
