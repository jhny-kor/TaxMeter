# 광주시 아동치과 주치의 지원 (광주광역시)

광주광역시 관할 지자체 지원금입니다. 경제적 어려움으로 치과 진료를 받기 힘든 아동에게 치과 주치의를 연계해 정기검진과 치료를 지원합니다. 충치 예방과 구강 건강을 지켜 아이들이 건강하게 성장할 수 있도록 돕습니다.
