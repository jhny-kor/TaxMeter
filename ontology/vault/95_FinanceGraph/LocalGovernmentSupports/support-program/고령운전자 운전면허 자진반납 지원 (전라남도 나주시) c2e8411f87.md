# 고령운전자 운전면허 자진반납 지원 (전라남도 나주시)

전라남도 나주시 관할 지자체 지원금입니다. 고령 운전자가 안전하게 운전을 중단할 수 있도록 지원금을 제공합니다. 자진반납 1인당 최대30만 원을 지역상품권으로 지급하여 교통사고 예방과 생활 안정에 도움을 줍니다.
