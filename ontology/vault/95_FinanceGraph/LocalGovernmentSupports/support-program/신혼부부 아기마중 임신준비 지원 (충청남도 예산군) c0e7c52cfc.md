# 신혼부부 아기마중 임신준비 지원 (충청남도 예산군)

충청남도 예산군 관할 지자체 지원금입니다. 신혼부부가 건강하게 임신과 출산을 준비할 수 있도록 지원하는 제도입니다. 
임신 준비 상담, 영양교육, 건강검진 지원 등이 제공되며, 신청 시 1회 참여 가능합니다. 
부부는 임신 초기 건강관리 부담을 줄이고, 안정적으로 출산을 준비할 수 있습니다.
