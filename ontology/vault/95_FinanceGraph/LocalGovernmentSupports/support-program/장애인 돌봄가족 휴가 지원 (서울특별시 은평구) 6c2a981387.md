# 장애인 돌봄가족 휴가 지원 (서울특별시 은평구)

서울특별시 은평구 관할 지자체 지원금입니다. 장애인을 돌보는 가족에게 휴식을 제공하기 위해 대체 돌봄 서비스와 휴가비를 지원합니다. 돌봄 부담을 완화하고 가족의 건강과 삶의 질을 지켜주는 제도입니다. 가족 구성원 모두가 더 나은 생활을 할 수 있도록 돕습니다.
