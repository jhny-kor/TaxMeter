# 시설원예재배환경개선사업 (충청남도 예산군)

충청남도 예산군 관할 지자체 지원금입니다. 시설원예 농가의 생산 환경을 개선하고 생산성을 높이기 위해 온·습도 조절, 관수·배수 시설, 환기장치 등 재배환경 개선 장비를 지원합니다. 농가 소득 증대와 안정적 작물 생산에 기여합니다.
