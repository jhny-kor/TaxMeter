# 축산 악취저감제 공급(한우·젖소) (전라남도)

전라남도 관할 지자체 지원금입니다. 한우와 젖소를 사육하는 농가가 축산 과정에서 발생하는 악취로 인한 불편과 환경 문제를 줄일 수 있도록 악취저감제를 지원합니다. 지원받은 농가는 쾌적한 축사 환경을 조성하고 주변 주민과 함께 안전하고 청결한 농장 운영이 가능합니다.
