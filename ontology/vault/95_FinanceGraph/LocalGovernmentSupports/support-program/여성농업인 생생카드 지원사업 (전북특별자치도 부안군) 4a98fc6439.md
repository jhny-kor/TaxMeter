# 여성농업인 생생카드 지원사업 (전북특별자치도 부안군)

전북특별자치도 부안군 관할 지자체 지원금입니다. 여성 농업인이 필요한 농업자재, 생활용품 등을 구매할 수 있도록 지원금을 지급하는 제도입니다. 
연간 일정 금액의 지원금을 카드로 지급하며, 사용처는 지정된 농업·생활 관련 상점입니다. 
여성 농업인은 생활비 부담을 줄이고, 생산 활동에 필요한 물품을 쉽게 구입할 수 있습니다.
