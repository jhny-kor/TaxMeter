# 출산장려금 지원 (전라남도 목포시)

전라남도 목포시 관할 지자체 지원금입니다. 출생일 기준 목포시에 주민등록을 두고 거주하는 출산가정에게 첫째 150만원, 둘째 250만원(출생신고 시 150만원 + 다음해 생일달 100만원), 셋째 350만원(출생신고 시 150만원 + 2년간 생일달 100만원), 넷째 450만원(출생신고 시 150만원 + 3년간 생일달 100만원), 다섯째 이상 550만원(출생신고 시 150만원 + 4년간 생일달 100만원)을 지원하는 제도입니다.
