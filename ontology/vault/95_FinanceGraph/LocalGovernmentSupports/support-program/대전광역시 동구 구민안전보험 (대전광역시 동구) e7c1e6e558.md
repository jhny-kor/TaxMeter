# 대전광역시 동구 구민안전보험 (대전광역시 동구)

대전광역시 동구 관할 지자체 지원금입니다. 동구민이 재난·사고·질병으로 인한 생활 불안을 최소화할 수 있도록 지원하는 제도입니다. 
보장 항목별로 최대 1천만 원까지 보험금 지급이 가능하며, 연간 자동 갱신됩니다. 
주민은 위급 상황 발생 시 경제적 부담을 줄이고, 안정적인 생활을 유지할 수 있습니다.
