# 신혼부부 결혼 축하 바우처 지원 (대구광역시 달성군)

대구광역시 달성군 관할 지자체 지원금입니다. 신혼부부의 새로운 시작을 응원하고 초기 생활비 부담을 완화하기 위해 지원하는 제도입니다.
