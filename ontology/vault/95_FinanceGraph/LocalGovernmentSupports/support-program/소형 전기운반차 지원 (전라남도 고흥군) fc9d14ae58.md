# 소형 전기운반차 지원 (전라남도 고흥군)

전라남도 고흥군 관할 지자체 지원금입니다. 농사일이나 작업 현장에서 효율적으로 짐을 운반할 수 있도록 소형 전기운반차를 지원합니다. 체력 부담을 줄이고 작업 시간을 단축하여 농업 생산성을 높일 수 있습니다. 장비 구입 부담을 완화해 안정적인 농업 환경을 조성합니다.
