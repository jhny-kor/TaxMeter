# 가족친화적 사회환경 조성사업 (대구광역시 달성군)

대구광역시 달성군 관할 지자체 지원금입니다. 가족 친화적 정책과 환경을 조성하여 주민이 일과 가정을 조화롭게 유지할 수 있도록 지원하는 제도입니다. 
가족친화 교육, 캠페인, 상담 서비스, 근로 환경 개선 지원 등을 제공합니다. 
주민은 가족 간 유대 강화와 삶의 질 향상 효과를 체감할 수 있습니다.
