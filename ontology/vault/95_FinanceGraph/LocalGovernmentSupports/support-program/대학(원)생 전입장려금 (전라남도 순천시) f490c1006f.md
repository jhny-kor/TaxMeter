# 대학(원)생 전입장려금 (전라남도 순천시)

전라남도 순천시 관할 지자체 지원금입니다. 대학이나 대학원에 진학하면서 다른 지역으로 전입하는 학생들에게 안정적인 생활 정착을 돕기 위해 장려금을 지원하는 제도입니다. 학업 준비와 생활 초기 비용 부담을 줄여, 학생이 학업에 집중할 수 있도록 돕습니다.
