# 진도군 산후조리비 지원 (전라남도 진도군)

전라남도 진도군 관할 지자체 지원금입니다. 출산 직후 산모의 회복과 아기 돌봄을 돕기 위해 산후조리비를 지원해요. 출생아 1인당 80만 원을 모바일 지역상품권으로 지급하며, 쌍둥이 출산 시에는 추가 지원도 있어요. 병원 조리비, 진료비, 건강보조식품 구입 등에 활용 가능합니다.
