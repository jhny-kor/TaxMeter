# 임신·출산 건강관리 비용지원 (충청남도 공주시)

충청남도 공주시 관할 지자체 지원금입니다. 임신 전후 산모의 건강을 체계적으로 관리하기 위해 산전검사, 기형아 검사, 영양제 구입, 산후검진 등 필요한 의료비를 지원합니다. 임신 초기부터 출산 후까지 건강관리를 돕고, 안전한 출산 환경을 조성합니다.
