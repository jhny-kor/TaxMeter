# 정선군민 안전보험 (강원특별자치도 정선군)

강원특별자치도 정선군 관할 지자체 지원금입니다. 정선군 주민이 재난, 사고, 질병 등 긴급 상황에서 보호받을 수 있도록 지원하는 제도입니다. 
보장 항목별로 최대 1천만 원까지 보험금 지급이 가능하며, 연간 자동 가입으로 편리합니다. 
주민은 사고 발생 시 경제적 부담을 줄이고, 안정적인 생활을 유지할 수 있습니다.
