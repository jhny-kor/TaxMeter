# 인천광역시 임산부 교통비 지원 (인천광역시)

인천광역시 관할 지자체 지원금입니다. 임산부가 대중교통이나 택시 이용 시 쓸 수 있는 교통비 포인트 50만 원을 지급해요. 안전하고 편리하게 이동할 수 있도록 돕는 출산 친화적 제도입니다.
