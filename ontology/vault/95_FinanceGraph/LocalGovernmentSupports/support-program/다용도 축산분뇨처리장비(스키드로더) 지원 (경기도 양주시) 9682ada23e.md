# 다용도 축산분뇨처리장비(스키드로더) 지원 (경기도 양주시)

경기도 양주시 관할 지자체 지원금입니다. 축산 농가가 분뇨 처리 및 농장 작업을 효율적으로 수행할 수 있도록 스키드로더와 소형굴삭기를 지원합니다. 지원 단가는 3,000만원/대로, 비용은 보조 50%, 자부담 50%입니다. 작업 효율과 환경 개선에 실질적인 도움이 됩니다.
