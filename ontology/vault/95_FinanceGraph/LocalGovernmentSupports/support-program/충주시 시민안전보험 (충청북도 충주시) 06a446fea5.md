# 충주시 시민안전보험 (충청북도 충주시)

충청북도 충주시 관할 지자체 지원금입니다. 충주시민이 재난, 사고, 화재, 폭발, 강도 등으로 피해를 입었을 경우 자동으로 보험금이 지급되는 제도입니다. 별도 가입 절차 없이 전 시민이 자동 가입되어 안전망을 강화합니다.
