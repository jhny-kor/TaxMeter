# 전입학생지원금 (경상남도 하동군)

경상남도 하동군 관할 지자체 지원금입니다. 다른 지역에서 이사 온 학생들의 안정적인 학교 적응을 돕기 위해 마련된 제도입니다. 학용품비, 교복비, 급식비 등을 지원해 학부모의 부담을 줄이고 학생들이 학업에 집중할 수 있도록 돕습니다. 최대 50만 원까지 지원됩니다.
