# GAP 인증 농가수수료 지원 (전북특별자치도 부안군)

전북특별자치도 부안군 관할 지자체 지원금입니다. 농산물의 안전성과 품질 향상을 위해 GAP 인증을 받은 농가의 인증 수수료를 지원합니다. 연 1회, 최대 50만 원까지 지원되어 인증 비용 부담을 줄이고, 소비자 신뢰 확보와 시장 경쟁력 향상에 기여합니다.
