# 축산농가 소독시설 지원 (전라남도 함평군)

전라남도 함평군 관할 지자체 지원금입니다. 가축 질병 예방과 위생 관리 강화를 위해 축산농가에 소독시설 설치를 지원합니다. 1농가당 최대 200만 원 지원으로 질병 예방과 생산성 향상에 도움을 줍니다.
