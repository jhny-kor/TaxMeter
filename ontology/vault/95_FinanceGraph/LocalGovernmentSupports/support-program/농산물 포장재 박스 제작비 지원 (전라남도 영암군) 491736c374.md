# 농산물 포장재 박스 제작비 지원 (전라남도 영암군)

전라남도 영암군 관할 지자체 지원금입니다. 영암군 농특산물 공동브랜드 사용 승인을 받은 농가·업체를 대상으로, 공동브랜드 상표도안을 인쇄한 포장재(박스) 제작비를 지원해 판로 경쟁력을 높이는 사업으로, 도지사품질인증·기찬들영암몰 입점 농가는 우선 지원되며, 비닐·스티로폼·공동브랜드 미표기·부당사용 이력 농가는 제외됩니다.
