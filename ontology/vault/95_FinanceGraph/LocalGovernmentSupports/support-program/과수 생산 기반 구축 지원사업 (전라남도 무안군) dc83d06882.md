# 과수 생산 기반 구축 지원사업 (전라남도 무안군)

전라남도 무안군 관할 지자체 지원금입니다. 과수 농가가 안정적으로 과수를 재배할 수 있도록 기반 시설과 장비를 지원합니다. 관수시설, 재배 장비, 교육 등을 제공하여 생산 효율을 높입니다. 과수 농가의 경쟁력 향상과 안정적인 소득 확보에 도움을 주는 제도입니다.
