# 출산장려금 지원 (경기도 양평군)

경기도 양평군 관할 지자체 지원금입니다. 양평군에 거주하는 가정을 대상으로 아기 출산 시 단계별로 장려금을 지원합니다. 첫째아·둘째아는 각 500만 원(4년간 분할), 셋째아는 1,000만 원(4년간 분할), 넷째아 이상은 최대 2,000만 원(5년간 분할)으로 지급되어 안정적인 양육과 가계 지원에 큰 보탬이 됩니다.
