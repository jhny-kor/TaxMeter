# 노인 성인용 보행기 지원 (충청북도 괴산군)

충청북도 괴산군 관할 지자체 지원금입니다. 거동이 불편한 노인의 안전한 이동과 일상생활 편의를 위해 보행기를 지원하는 제도입니다.
