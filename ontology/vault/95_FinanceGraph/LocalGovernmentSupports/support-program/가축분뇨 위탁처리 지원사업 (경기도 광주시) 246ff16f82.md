# 가축분뇨 위탁처리 지원사업 (경기도 광주시)

경기도 광주시 관할 지자체 지원금입니다. 가축분뇨 처리 비용 부담을 줄이고 환경오염을 예방하기 위해 위탁처리 비용을 지원하는 제도입니다. 
분뇨 위탁처리 비용의 최대 70%를 지원하며, 연 1회 신청 가능합니다. 
농가는 처리 비용 부담을 줄이고, 환경을 보호하며 농장을 깨끗하게 관리할 수 있습니다.
