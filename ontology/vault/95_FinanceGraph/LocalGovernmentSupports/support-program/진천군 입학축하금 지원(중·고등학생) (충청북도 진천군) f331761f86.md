# 진천군 입학축하금 지원(중·고등학생) (충청북도 진천군)

충청북도 진천군 관할 지자체 지원금입니다. 진천군에 주소를 둔 신입생에게 입학을 축하하는 의미로 1인당 10만 원을 지원해요. 유치원부터 고등학교까지 해당돼 학부모의 초기 학비 부담을 줄이는 데 도움이 됩니다.
