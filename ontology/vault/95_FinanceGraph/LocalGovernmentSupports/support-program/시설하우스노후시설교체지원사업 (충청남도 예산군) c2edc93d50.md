# 시설하우스노후시설교체지원사업 (충청남도 예산군)

충청남도 예산군 관할 지자체 지원금입니다. 시설하우스의 노후화로 인한 생산 효율 저하와 안전사고를 예방하기 위해 구조물, 비닐, 온·습도 조절 장치 등 주요 시설을 교체·보수할 수 있도록 지원합니다. 농가의 작업 환경 개선과 생산성 향상에 도움을 주며, 안정적인 농업경영을 지원합니다.
