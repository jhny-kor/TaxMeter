# 성남시 시민안전보험 (경기도 성남시)

경기도 성남시 관할 지자체 지원금입니다. 성남시민이 예상치 못한 사고와 재난으로부터 보호받을 수 있도록 안전보험에 가입됩니다. 사고 발생 시 치료비와 보상을 지원하여 생활 안정에 도움을 줍니다. 시민이 안심하고 생활할 수 있는 환경을 제공합니다.
