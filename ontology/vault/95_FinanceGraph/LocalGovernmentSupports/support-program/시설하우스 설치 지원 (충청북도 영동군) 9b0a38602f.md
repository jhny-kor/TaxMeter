# 시설하우스 설치 지원 (충청북도 영동군)

충청북도 영동군 관할 지자체 지원금입니다. 농업 생산성을 높이고 농작물 생육 환경을 개선하기 위해 시설하우스 설치 비용 일부를 지원하는 제도입니다. 
설치 1회당 일정 금액을 지원하며, 농가는 안정적인 농작물 생산과 효율적인 농업 경영이 가능합니다.
