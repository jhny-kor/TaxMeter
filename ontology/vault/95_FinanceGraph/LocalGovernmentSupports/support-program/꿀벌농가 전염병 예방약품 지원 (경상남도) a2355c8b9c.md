# 꿀벌농가 전염병 예방약품 지원 (경상남도)

경상남도 관할 지자체 지원금입니다. 양봉 농가에서 꿀벌 전염병 예방을 위해 필요한 약품을 지원하는 제도입니다. 
질병 발생 위험을 줄이고 안전한 양봉 생산 환경을 조성하여 농가 소득 안정에 기여합니다.
