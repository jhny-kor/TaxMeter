# 찰옥수수 명품화 (강원특별자치도 영월군)

강원특별자치도 영월군 관할 지자체 지원금입니다. 찰옥수수 농가가 고품질 옥수수를 생산하고 판로를 확대할 수 있도록 생산기술, 농업자재, 컨설팅을 지원하는 제도입니다. 상품성을 높여 농가 소득 증대와 지역 특산물 경쟁력 향상에 기여합니다.
