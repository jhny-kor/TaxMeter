# 근로복지공단 생활안정자금 대출이자 지원 (경상남도 창원시)

경상남도 창원시 관할 지자체 지원금입니다. 근로복지공단에서 생활안정자금을 대출받은 근로자에게 이자 일부를 지원합니다.
