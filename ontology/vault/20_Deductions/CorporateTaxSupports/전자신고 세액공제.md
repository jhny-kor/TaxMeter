# 전자신고 세액공제

법인이 직접 법인세를 전자신고할 때 적용되는 세액공제입니다.

## 근거 조항

조세특례제한법 제104조의8

## 기준 내역

- **전자신고 세액공제**: 기준 유형: deduction; 기준항목: 법인세 전자신고; 선정기준 분류: official-standard; 조건: 법인이 직접 전자신고; 선정기준 설명: 해당 제도에서 대상 여부, 세율, 공제액, 한도 또는 신고기한을 판정하기 위해 공식 출처가 사용하는 기준항목입니다.; 확인 방법: 각 criterion의 출처 노드와 관련 신고·신청 서류에서 확인합니다.; 선정 규칙: 조건 문구와 구조화된 금액·비율·기간 필드를 함께 보고 해당 구간 또는 요건을 선택합니다.; 근거 조항: 조세특례제한법 제104조의8; 공제액: 10,000원; 출처: [[90_Sources/법인세 공제감면|법인세 공제감면]]
