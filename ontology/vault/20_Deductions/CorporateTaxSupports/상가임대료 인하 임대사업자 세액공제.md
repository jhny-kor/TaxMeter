# 상가임대료 인하 임대사업자 세액공제

상가건물 임대료 인하액에 적용되는 세액공제입니다.

## 기준 내역

- **임대료 인하액**: 기준항목: 상가건물 임대료 인하액; 조건: 소상공인 임차인 임대료 인하 등 요건 충족; 세액공제율: 70%; 비고: 요건과 과세연도별 적용기한 확인 필요; 출처: [[90_Sources/법인세 공제감면|법인세 공제감면]]
